# Changelog

All notable changes to TestData Factory will be documented in this file.

## 1.0.0 - 2026-07-18

- Added Java, TypeScript, and Python page-object import for common Selenium and Playwright page classes.
- Added optional local-first dual-agent AI scenario assist with generator and validator feedback through CLI and API.
- Expanded generated scenarios with security, robustness, boundary, uniqueness, and cross-field coverage.
- Added dependency-aware generation for confirmation fields, date ranges, and numeric min/max pairs.
- Brought Python, Java, and TypeScript local generators to strategy parity.
- Updated the self-hosted API, OpenAPI spec, examples, docs, and release acceptance benchmark for the stable Release 1 surface.

## 0.1.0 - 2026-07-17

- Prepared the Python engine and CLI package for an initial PyPI release as `testdata-factory-engine`.
- Prepared the Java SDK package for an initial Maven Central release as `io.github.iisleem.testdatafactory:testdata-factory`.
- Prepared the TypeScript SDK package for an initial npm release as `testdata-factory`.
- Included the shared contract schema, OpenAPI spec, and example contracts for first-release validation.
